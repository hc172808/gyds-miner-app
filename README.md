# GYDS Miner & Wallet App

This project provides the **mobile (React Native) and browser CPU miner** for the GYDS ecosystem.  
It includes a **wallet, mining engine, price oracle integration, off-chain reward tracking**, and is ready to connect to your GYDS full node.

---

## Features

- React Native **mobile app** (Android + iOS)
- **Browser CPU miner**
- **HD wallet** with seed phrase, address, and transaction signing
- Mining engine with **120-second block target** and dynamic difficulty
- **Price oracle integration**
- Track **off-chain mining rewards**
- Send/receive **GYDS/GYD tokens**
- Swap-ready architecture (AMM / liquidity pool)
- Works fully with your **off-chain hub**
